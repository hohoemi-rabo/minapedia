# 09: リアクション機能

> マイルストーン: M3 | 優先度: 中

## 概要

3種のリアクションボタン。いいね数は非表示または控えめ表示。
1ユーザー1投稿につき各タイプ1回まで（トグル式）。

## リアクション種類

| タイプ     | DB値      | 表示       |
| :--------- | :-------- | :--------- |
| すてき     | `suteki`  | すてき     |
| 行ってみたい | `ikitai`  | 行ってみたい |
| 参考になった | `sanko`   | 参考になった |
