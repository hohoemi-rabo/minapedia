# 06: 投稿カードコンポーネント

> マイルストーン: M2 | 優先度: 高
> Status: DONE

## 概要

写真あり・なし両対応の投稿カード。写真なし時はテーマ別アイコン＋背景カラーで表示。

## カード構成

```
[写真 or テーマアイコン＋背景]
[テーマバッジ] [エリアタグ]
タイトル（大きく・太字）
ニックネーム
本文（2行プレビュー）
[続きを読む]
すてき / 行ってみたい / 参考になった
```

## テーマ別デザイン対応表

| テーマ             | アイコン | 背景カラー         |
| :----------------- | :------- | :----------------- |
| よかった場所       | 地図     | ナチュラルグリーン |
| おいしいもの       | フォーク | ウォームオレンジ   |
| きれいな景色       | 花       | ソフトピンク       |
| また行きたい       | スター   | ゴールドイエロー   |
| 行ってみたいところ | ピン     | スカイブルー       |
| やってみたいこと   | スパークル | ラベンダー       |
