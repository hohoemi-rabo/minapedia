# 12: 管理画面 — おすすめ固定表示

> マイルストーン: M4 | 優先度: 中

## 概要

管理者がホーム画面上部に表示する「今週のおすすめ投稿」を最大3件まで選択・固定する機能。

## 機能

- published 投稿からおすすめを選択
- `is_featured = true` をトグル
- 最大3件の制限
- ホーム画面の先生おすすめセクションに反映
