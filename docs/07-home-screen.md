# 07: ホーム画面（みんなの新着）

> マイルストーン: M3 | 優先度: 高

## 概要

トップページ。新着投稿の一覧表示とテーマ絞り込み。
無限スクロールは使わず「もっと見る」ボタンで5件ずつ追加読み込み。

## 画面構成（上から順）

1. 安心メッセージ（初回のみ）: 「このサービスは教室の生徒さんだけが見られます」
2. 先生おすすめ（最大3件・固定表示）
3. テーマで絞る（テーマボタン）
4. みんなの新着（5件ずつ、「もっと見る」ボタン）
