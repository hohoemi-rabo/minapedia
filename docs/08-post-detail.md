# 08: 投稿詳細画面

> マイルストーン: M3 | 優先度: 高
> Status: DONE

## 概要

投稿カードから遷移する詳細表示画面。写真のスワイプ切り替え、全文表示。
自分の投稿は編集・削除ボタンを表示。

## 表示要素

- 写真（最大3枚・スワイプ切り替え）or テーマアイコン＋背景
- タイトル・ニックネーム・エリア・テーマ
- 全文表示（どこがよかった？ ＋ ひとこと感想）
- 場所の名前
- リアクションボタン
- 自分の投稿のみ: 編集・削除ボタン
- 安心メッセージ: 「あとで自分で編集・削除できます」
